📌 Project Overview

This project analyzes sales data to identify revenue trends, profit performance, and regional sales distribution using Python and data visualization techniques.

🛠 Tools Used

  • Python (Pandas, Matplotlib)
  • CSV Dataset
  • Power BI Dashboard
  • SQL

📊 Key Insights

  • West region generates the highest profit.
  • Electronics category contributes highest revenue.
  • Fashion category has the highest profit margin.
  • High revenue products don’t always give high profit.
  • Repeat customers increase total sales.
